# Photography Portfolio

Minimalistic photography portfolio website built with React and Vite.

Live demo: [[your-link-here]](https://foto-portfolio-two.vercel.app/)

---

## About

This project is a personal photography portfolio designed to showcase visual work in a clean and elegant style.

The focus is on:

* natural light
* soft colors
* simple composition
* emotional storytelling

The layout is intentionally minimal to keep attention on the images.

---

## Features

* Responsive layout (desktop + mobile)
* Clean grid-based portfolio
* Hero section with overlay
* About, Services, Pricing, Contact sections
* Environment-based external links (e.g. Instagram)
* Fast performance via Vite

---

## Tech Stack

* React
* JavaScript
* Vite
* Tailwind CSS

---

## Project Structure

```
/public
  /images
/src
  App.jsx
  index.css
```

Images are stored in `/public/images` and rendered dynamically.

---

## Installation

Clone the repository:

```
git clone [https://github.com/your-username/your-repo.git](https://github.com/thePolly/foto-portfolio.git)
```

Install dependencies:

```
npm install
```

Run locally:

```
npm run dev
```

---

## Environment Variables

Create a `.env` file:

```
VITE_INSTAGRAM_URL=https://www.instagram.com/your-profile/
```

---

## Customization

To update content:

* Replace images in `/public/images`
* Edit portfolio items inside `App.jsx`
* Update text in About / Services / Contact sections
